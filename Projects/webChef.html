Project Status

Planned

Project Concept

A tool that allows users to sketch a website as if they were using MS Paint, and transforms those sketches into functional web apps.

Project Goal

The goal behind webchef is to simplify web design for non-developers and enhance creativity in UI/UX prototyping.

Key Features

Intuitive Drawing Interface

Users can sketch web pages as if using MS Paint, with drawing tools like lines, shapes, text, and colors.

Instant Code Generation

Transforms sketches into functional HTML, CSS, and JavaScript code.
