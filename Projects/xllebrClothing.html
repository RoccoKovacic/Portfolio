Project Overview

Xllebr Clothing Brand is an Australian streetwear brand, a personal project created in 2024 and currently in development. Fashion is one of my main personal interests and i am using this brand to create clothes that i enjoy and hopefully others do too.

What Have I Learned?

From the inital planing phase all the way to the building phase, I have developed and strengthened my skills in frontend development. Nearing the end of this project I am now learning about how payment systems work and how to connect those to my own E-Commerce platfrom. Aswell as this I am also developing skills in creating my own databases for user logins and account creation, as well as logging user transaction history.
